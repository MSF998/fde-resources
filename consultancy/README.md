# Consultancy Track

One session per week covering AI solution design, stakeholder management, business case building, and client communication.

Each session folder (`XX-topic-name/`) contains:
- `pre-read.md` — notes on preparatory material
- `notes.md` — class notes and frameworks
- `post-read.md` — post-session reflection and synthesis
- `resources.md` — links to relevant tools and references
