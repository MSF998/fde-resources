# Technical Track

Two sessions per week covering AI/ML engineering: LLMs, RAG pipelines, agents, infrastructure, and more.

Each session folder (`XX-topic-name/`) contains:
- `pre-read.md` — notes on preparatory material
- `notes.md` — class notes and key concepts
- `post-read.md` — post-session reflection and synthesis
- `resources.md` — links to relevant tools, papers, and docs
