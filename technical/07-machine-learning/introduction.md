## Key Concepts

- “A computer program is said to learn from experience E with respect to some task T and some performance measure P, if its performance on T, as measured by P, improves with experience E.”This classic definition (from his seminal 1997 textbook, Machine Learning) breaks learning into three distinct components:Task (T): The specific problem or action you want the program to accomplish (e.g., classifying emails as spam, translating languages, or steering a car).Performance Measure (P): The metric used to evaluate how well the computer executes the task (e.g., accuracy percentage, mean squared error, or profit).Experience (E): The data or history the program uses to learn and adjust its approach (e.g., a dataset of historical housing prices or past customer transactions).
